# Lane 00 - UI/UX Portal Entrance Preview

Branch / worktree: `worktree/00-uiux-portal-entrances`

## Purpose

This lane exists only to verify the Steel Craft visual direction and the beginning entrance of each portal before deeper functionality is built.

## Included in this lane

- Landing page
- Simulated authentication / role-selection entrance
- Admin Portal entrance
- HR Portal entrance
- Projects Portal entrance
- Sales & Marketing Portal entrance
- Vendor Portal entrance
- Steel Craft luxury UI direction using black, iron red / burgundy, white, and dark metallic panels

## Not included yet

- Real authentication backend
- Database wiring
- Employee records
- Project/job management logic
- Sales CRM logic
- Vendor document workflows
- Admin permission management

## Approval gate

The client should review only visual direction, branding, portal entrances, layout, and first impression. After visual approval, this lane should be hardened before the next lane begins.
