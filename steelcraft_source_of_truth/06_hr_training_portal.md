# HR, Employee / Employer Room, and Training Portal Source of Truth

## Correction made

All employees are salary. Remove the time clock completely. The portal should not include punch-in/punch-out, shift tracking, payroll time capture, or time-clock widgets.

## Primary rooms

### Employee / Employer Room

Employee-facing room where employees can view their HR information, request PTO, access the handbook, raise HR concerns, and complete training.

Required screens/sections:

- Employee profile
- Start date
- Anniversary date or calculated work anniversary
- PTO balance
- PTO tracking and request history
- Time-off request form
- Time-off request approval status
- HR concerns / complaints / employee issue submission
- Employee handbook access
- Employee handbook acknowledgement/signature
- Training module room
- Assigned training modules
- Training completion status
- Employee documents

### HR Admin Room

Employer/admin room where management can maintain employee records, approve time off, manage handbook versions, review concerns, and assign training.

Required screens/sections:

- Employee directory
- Employee record detail
- Start date and anniversary tracking
- PTO balance management
- PTO/time-off request approval queue
- HR concerns/complaints inbox
- Concern status: new / under review / resolved / closed
- Private admin notes for concerns
- Employee handbook upload/versioning
- Handbook acknowledgement tracking
- Training course/module management
- Training assignment and completion tracking

## PTO tracker

Need screens for:

- Employee PTO balance
- PTO request form
- Manager/admin approval
- Calendar view
- Request history
- Remaining PTO

## Time-off requests

Need fields for:

- Employee
- Request type: PTO / sick / unpaid / other
- Start date
- End date
- Number of hours or days
- Reason / notes
- Status: pending / approved / denied / cancelled
- Manager/admin reviewer
- Reviewed timestamp
- Admin notes

## HR concerns / complaints

Need a private intake area where employees can submit concerns, complaints, or HR issues.

Need fields for:

- Employee submitting concern
- Concern category
- Description
- Optional attachment
- Confidential flag
- Status: submitted / under review / resolved / closed
- Assigned reviewer
- Admin notes
- Resolution notes
- Created timestamp
- Updated timestamp
